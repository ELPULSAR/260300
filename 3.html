APIs Gratuitas de IA Generativa

OpenAI (GPT-3.5)

Acceso gratuito con límite mensual de tokens usando una cuenta gratuita. Permite generación de texto, conversación y análisis.

Hugging Face Inference API

Permite usar modelos como GPT-2, BLOOM, T5, etc. gratuito con limitaciones en el uso por minuto. Muy versátil.

Cohere

API enfocada en NLP generativo y embeddings. Plan gratuito con acceso limitado diario y ejemplos claros.

Replicate

Ejecuta modelos de texto e imagen generativa (como Stable Diffusion, LLaMA, etc.) directamente desde API. Tiene créditos gratuitos.

Stability AI

Ofrece generación de imágenes con Stable Diffusion desde su API. Cuenta gratuita con límites mensuales.

DeepAI

API para generación de texto, imagen y más. Acceso gratuito con clave API sencilla y sin mucha verificación.

Perplexity AI API

Permite hacer preguntas y generar respuestas con contexto real y búsqueda en vivo. Plan gratuito disponible.
